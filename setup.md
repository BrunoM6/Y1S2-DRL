## How to setup
0. Data is presumed to be in 'data/' folder, with each one of the four original folders and names.
1. use venv to create virtual environment `python3 -m venv .venv`
2. enter virtual environment`source .venv/bin/activate`.
3. install requirements on environment `pip install -r -requirements.txt`.
4. run project on notebook after selecting appropriate kernel.
